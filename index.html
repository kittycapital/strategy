Strategy(MSTR) 비트코인 매입 대시보드 안내

이 대시보드는 SEC EDGAR의 무료 API를 활용하여 Strategy Inc(구 MicroStrategy, 티커: MSTR)의 비트코인 매입 이력을 자동으로 추적합니다. Strategy는 2020년 8월부터 기업 재무전략으로 비트코인을 매입하기 시작했으며, 현재 세계 최대의 기업 비트코인 보유자입니다.

데이터 소스

모든 데이터는 SEC EDGAR의 8-K(임시보고서) 공시에서 추출됩니다. Strategy는 비트코인 매입 시 Item 8.01 또는 Item 7.01을 통해 매입 수량, 총 매입가, 평균 단가를 공개합니다. SEC EDGAR Submissions API(data.sec.gov)는 무료이며 API 키가 필요하지 않습니다.

자금 조달 방식

Strategy는 다양한 방식으로 비트코인 매입 자금을 조달합니다: ATM(At-The-Market) 프로그램을 통한 보통주 발행, 전환사채(Convertible Notes) 발행, 우선주(STRK, STRF, STRD, STRC) 발행, 기존 보유 현금, 담보 대출 등이 있습니다. 2024년 하반기부터는 ATM 주식 발행이 주요 자금원이 되었습니다.

주요 지표

BTC Yield(주당 BTC)는 주식 희석 대비 BTC 보유량 증가율을 나타내는 Strategy의 핵심 KPI입니다. mNAV(Market-value NAV)는 기업가치 대비 비트코인 가치 비율을 보여줍니다. 이 대시보드는 GitHub Actions를 통해 매일 자동 업데이트됩니다.
